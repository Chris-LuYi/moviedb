### Built With

* [React](https://reactjs.org/)
* [UmiJS](https://umijs.org/)
* [Ant Design](https://ant.design/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Chris-LuYi/moviedb.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the dev server
    ```sh
   npm run start
   ```
4. Build the project
    ```sh
   npm run build
   ```
